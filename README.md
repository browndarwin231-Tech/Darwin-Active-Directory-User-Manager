# Darwin Active Directory User Manager

A Python-based Active Directory simulation that allows administrators to create, search, disable, and manage user accounts through an interactive command-line interface.

## Features

- Create user accounts
- View all users
- Search for users
- Disable user accounts
- Simulates Active Directory administration tasks
- Beginner-friendly Python code

## How It Works

The application simulates common Active Directory tasks performed by Help Desk Technicians, IT Support Specialists, and System Administrators.

### Available Options

1. Create User
2. View Users
3. Disable User
4. Search User
5. Exit

## Example

```text
=== Active Directory User Manager ===

1. Create User
2. View Users
3. Disable User
4. Search User
5. Exit

Choose an option: 1

Enter username: jsmith

User Created
```
